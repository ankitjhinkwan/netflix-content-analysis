## 🚀 Getting Started

```bash
git clone https://github.com/ankitjhinkwan/twitter-sentiment-analysis
cd twitter-sentiment-analysis
pip install -r requirements.txt
cd data
python generate_data.py
cd ..
streamlit run app/app.py
```

## 🔑 Key Findings

- **Technology & Entertainment** generate most positive sentiment
- **Politics & Economy** dominate negative sentiment
- **Mobile** is the #1 device (55%+ share)
- **Evening hours** see peak activity
- **Negative tweets** get more engagement than positive ones
